# MeowCare

MeowCare is a RAG-powered cat care knowledge assistant for cat owners. The app is designed to answer questions about cat health, nutrition, and behavior using responses generated by Claude and grounded in a curated knowledge base with citations.

## Stack

- Next.js 14 with the App Router
- Tailwind CSS
- Supabase with pgvector
- Clerk for authentication
- Anthropic and OpenAI SDKs
- Vercel for deployment

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Copy the environment template and fill in your keys:

```bash
cp .env.local.example .env.local
```

3. Start the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000).

## Project Structure

- `app/`: App Router pages and layout
- `components/`: Shared UI components
- `lib/`: Utilities and service clients
- `scripts/`: Data ingestion and maintenance scripts
